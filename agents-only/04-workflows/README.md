# 工作流程索引（Workflows Index）

> 此目录包含重构相关的工作流程和追踪文档

---

## 流程文档

| 文档 | 用途 |
|------|------|
| [REFACTOR-PROGRESS.md](./REFACTOR-PROGRESS.md) | 各 crate 重构进度追踪 |
| [CODE-MIGRATION.md](./CODE-MIGRATION.md) | 代码迁移日志 |
| [BLOCKERS.md](./BLOCKERS.md) | 阻塞项清单 |
| [TEST-MAPPING.md](./TEST-MAPPING.md) | 测试映射 |
| [CRATE-DETAIL.md](./CRATE-DETAIL.md) | 各 crate 详细职责 |

---

## 使用方式

1. 重构开始前：检查 BLOCKERS.md
2. 移动代码时：记录到 CODE-MIGRATION.md
3. 完成阶段后：更新 REFACTOR-PROGRESS.md
4. 修改测试时：同步 TEST-MAPPING.md
