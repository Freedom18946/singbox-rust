# 99 验收清单总表（Checklist）

> 这是一张“可落地的验收表”。你可以把它贴到 Jira/Epics 或 PR 模板里：一条条打勾，系统就从“能跑”走向“可依赖”。

## A. 配置与工具链

- [ ] MUST：支持 JSON 配置与顶层结构（log/dns/ntp/certificate/endpoints/inbounds/outbounds/route/services/experimental）
- [ ] MUST：`sing-box check` 可定位错误路径
- [ ] MUST：`sing-box format` 可稳定格式化
- [ ] MUST：`sing-box merge` 可合并目录配置
- [ ] MUST：`sing-box rule-set compile` 生成 `.srs`
- [ ] SHOULD：`sing-box rule-set convert --type adguard` 可用

## B. 日志与可观测性

- [ ] MUST：level 过滤
- [ ] MUST：timestamp 开关
- [ ] MUST：output（console/file）
- [ ] MUST：disabled 全局关闭

## C. DNS 子系统

- [ ] MUST：DNS server 类型齐全（local/hosts/tcp/udp/tls/quic/https/h3/dhcp/fakeip/tailscale/resolved + legacy 路线）
- [ ] MUST：DNS Rule / DNS Rule Action 可用（含 predefined）
- [ ] SHOULD：FakeIP
- [ ] SHOULD：reverse_mapping + cache_capacity
- [ ] SHOULD：平台差异处理（Android/macOS 等）

## D. 路由与规则

- [ ] MUST：Route Rule 基础匹配（domain/ip/port/source 等）
- [ ] MUST：支持 logical 规则组合
- [ ] MUST：Rule Action（route + resolve）
- [ ] MUST：Protocol Sniff（协议与 QUIC client 识别）
- [ ] MUST：Rule-Set 加载与引用（source/binary/remote）

## E. 入站（Inbounds）

- [ ] MUST：direct/mixed/socks/http/shadowsocks/vmess/trojan/naive/hysteria/shadowtls/vless/tuic/hysteria2/anytls/tun/redirect/tproxy
- [ ] MUST：Listen Fields 复用
- [ ] MUST：透明代理能恢复 original destination
- [ ] SHOULD：sniff 与 auth_user 暴露给路由

## F. 出站（Outbounds）

- [ ] MUST：direct/block/socks/http/shadowsocks/vmess/trojan/naive/wireguard/hysteria/shadowtls/vless/tuic/hysteria2/anytls/tor/ssh/dns/selector/urltest
- [ ] MUST：Dial Fields 复用
- [ ] MUST：selector/urltest 动态选择闭环
- [ ] SHOULD：Multiplex / V2Ray Transport / UDP-over-TCP / TCP Brutal / Pre-match
- [ ] SHOULD：WireGuard outbound → endpoint 迁移路线

## G. Endpoint

- [ ] MUST：endpoints[] 可加载
- [ ] MUST：WireGuard/Tailscale endpoint（按文档导航）
- [ ] SHOULD：DERP 联动、tailscale DNS server 联动

## H. TLS 与证书

- [ ] MUST：Certificate store（system/mozilla/chrome/none）与证书加载/热重载
- [ ] MUST：TLS 基础字段（SNI/ALPN/版本）
- [ ] SHOULD：pinning / mTLS / ECH / Reality / fragment / kTLS
- [ ] SHOULD：ACME + DNS‑01（provider 全覆盖）

## I. Services & Experimental

- [ ] SHOULD：services[] 类型（derp/resolved/ssm_api/ccm/ocm）可启用并具备安全默认
- [ ] SHOULD：experimental.cache_file/clash_api/v2ray_api 可按需启用
- [ ] MUST：迁移与弃用提示明确可操作
