## 1. 测试隔离环境

### 背景
本机可能已运行真实 Go 版本 TUN 代理，直接测试会冲突。

### 选项

**A) 创建 test-server/ 子项目**
```
singbox-rust/
└── test-server/            # 模拟服务端（Rust）
    ├── Cargo.toml
    └── src/main.rs         # 简单的 Trojan/SS 服务端
```
- 优点：轻量、无外部依赖
- 缺点：需要编写和维护

**B) Docker 隔离**
```yaml
# docker-compose.yml
services:
  test-server:
    image: singbox-test-server
    ports: ["10080:10080"]
```
- 优点：完全隔离
- 缺点：需要 Docker 环境

**C) 使用公网测试服务器**
- 优点：真实环境
- 缺点：网络依赖、可能不可用

**D) 手动切换**
- 停止 Go 版本 → 测试 Rust 版本 → 恢复
- 优点：简单
- 缺点：不方便
