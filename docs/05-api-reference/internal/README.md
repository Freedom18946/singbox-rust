# Internal APIs

## Overview

Internal APIs are Rust traits and types used by adapters, router, and admin services.

## Where to look

- `crates/sb-core` for core traits and routing
- `crates/sb-adapters` for protocol adapters
- `crates/sb-admin-contract` for response envelope types

## Rustdoc

```bash
cargo doc --workspace --all-features --no-deps
```
