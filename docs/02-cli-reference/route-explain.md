# `route` Command

This page keeps its historical filename, but the live command is `route`.

Example:

```bash
cargo run -p app -- route -c examples/quick-start/01-minimal.json --dest example.com:443 --explain --with-trace
```

Important options:

- `--dest <DEST>`
- `--udp`
- `--format <human|json|sarif>`
- `--explain`
- `--with-trace`
