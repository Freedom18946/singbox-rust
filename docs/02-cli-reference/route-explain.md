# `route` Command

Explain routing decisions for a destination.

## Usage

```bash
singbox-rust route -c config.yaml --dest example.com:443 --explain
```

## Common Options

- `--dest <ADDR>`: destination host:port
- `--explain`: include rule match details
- `--format <text|json>`: output format
