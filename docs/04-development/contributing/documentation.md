# Documentation Guide

How to update docs in this repository.

## Where Docs Live

- `docs/` for user/dev/ops documentation
- `docs/README.md` is the top-level index

## Link Hygiene

- Prefer relative links
- Avoid linking to non-existent pages
- Run a quick audit when adding new pages
